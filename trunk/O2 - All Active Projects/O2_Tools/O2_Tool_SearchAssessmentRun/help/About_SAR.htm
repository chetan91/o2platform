Search Assessment Run (scrool down for help)

This module is designed to allow the quick filtering and analsis of saved assessment files analysis. Its key features include advanced Filtering capabilities, trace's visualization and creation of individual saved assessment files.

This module is not dependend on Ounce 6.x. (because it is only manipulating XML files)

To quick start click on the several 'Load Demo File from xxx' below, or drag & drop an previously generated file into the red 'Drop Saved Assessment Files Here box.

In case of problems you can:
  • Post a question on O2's website Online Support Forum
  • Submit a request directy to the O2 development team using the Ask O2 Support page
  • Send a bug report to the O2 team using the 'Report a Bug' Button (located on the bottom right of this module)
